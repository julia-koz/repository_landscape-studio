# 🌿 IL Nature - Landscape Design Studio

This is a full-stack web application for a professional landscape design studio. The project features a modern user interface, a portfolio of completed works, and an administrative panel for content management.

## 🚀 Live Demo
(https://repository-landscape.vercel.app/)

## ✨ Features
* **Dynamic Portfolio:** Showcase of landscape design projects with high-quality images.
* **Services Overview:** Detailed descriptions of design, planting, and maintenance services.
* **Admin Panel:** Secure dashboard to manage projects and site content.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
* **Contact Integration:** Easy way for clients to reach out for consultations.

## 🛠️ Tech Stack
* **Frontend:** React.js, CSS3, JavaScript (ES6+)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Deployment:** Vercel

## 📁 Project Structure
* `/frontend` - React application (UI/UX)
* `/backend` - Express server and API routes
